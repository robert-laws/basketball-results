# NBA Finals Results

A static website displaying historical NBA Finals data with detailed game-by-game results.

**Live Site:** https://robert-laws.github.io/basketball-results/

## Features

- Browse all NBA Finals from 2010-2024
- View detailed series information including:
  - Champion and runner-up teams
  - Series result (e.g., 4-2)
  - Finals MVP
  - Game-by-game scores and locations

## Tech Stack

- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- React Router for navigation
- GitHub Pages for hosting

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Data

NBA Finals data is stored in `src/data/finals.json` and can be extended with additional years or details.
